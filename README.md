📌 Project Overview

This project enhances customer experience by analyzing emotions using both text and audio inputs.
It combines Natural Language Processing (NLP) and Audio Signal Processing to achieve more accurate sentiment detection compared to traditional methods.

🎯 Objective
🔍 Detect customer emotions in real-time
🧠 Combine text sentiment + audio emotion
⚡ Identify mixed emotions
📈 Improve decision-making in customer support

🧠 Key Features
✨ Text Sentiment Analysis (Logistic Regression, SVM, BERT)
🔊 Audio Emotion Recognition (MFCC + ML Models)
🔗 Multimodal Fusion (Text + Audio)
⚡ Real-time Sentiment Prediction
📊 Interactive Dashboard
